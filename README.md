### 🎓 VKinder — бот для знакомств (командный проект)
[🔗 Lampropeltiss/VKinder](https://github.com/Lampropeltiss/VKinder)

Программа реализует функционал VK-бота для поиска кандидатов, подходящих для знакомства, и оценки их пользователем ВК. Бот взаимодействует с пользователем через личные сообщения в ВКонтакте, предоставляя интерфейс для просмотра профилей кандидатов и управления списком избранных.

**Мой вклад:** проектирование БД, класс `VkBotDatabase` (CRUD-методы), настройка логирования.
